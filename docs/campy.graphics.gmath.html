campy.graphics.gmath module¶

This file exports several functions for working with graphical gemoetry along with the mathematical constants PI and E

campy.graphics.gmath.E = 2.718281828459045¶

The mathematical constant e, which is the base of the natural logarithm

campy.graphics.gmath.PI = 3.141592653589793¶

The mathematical constant pi, which is the ratio of the circumference of a circle to its diameter

campy.graphics.gmath.vector_angle(x, y)[source]¶

Returns the angle in degrees from the origin to the specified point. This functoin takes account of the fact that the graphics coordinate system is flipped in the y direction from the traditional Cartesian plane.

@type pt: GPoint @param pt: GPoint to angle to, will override x and y parameters @type x: float @param x: x value of point @type y: float @param y: y value of point @rtype: float @return: degrees
